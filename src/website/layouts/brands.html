How to be featured here

Is your company using fastify? That's great!

If you would like your company to be featured in this list, we would love it too!

Here's how to submit your company, your logo and other few relevant details, so that we can update this page:

  1. Fork the Fastify website repository on GitHub.
  2. Add the logo in the folder src/website/content/images/brands.
    Please make sure that your logo complies to the following specs:
    • Encoded in SVG (Scalable Vector Graphics) format
    • Use only 2 colors: #fff (white) and #999 (gray)
    • Be reasonably optimized/minified (you can use svgminify)
  3. Update the file src/website/data/brands.yml with a new entry (in alphabetical order by company name).
    Your entry must provide:
    • name: the name of your company
    • image: the filename of the file you added in the previous step
    • link: a link to a website that represents your organisation
    Please, make sure not to break the YAML syntax of the file.
  4. Submit a PR!

As soon as your PR is reviewed and merged, your awesome logo will appear in this page.
